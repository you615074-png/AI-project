申请前提

  • 一个 GitHub 账号
  • 学校的 .edu.cn 邮箱（如 2023001@xxx.edu.cn）
  • 学生证（备用）

申请步骤（5分钟）

  1. GitHub Settings → Profile → 填真实姓名和学校（英文）
  2. 打开 education.github.com/pack → Get your pack
  3. 选 Student → 填 .edu.cn 邮箱 → 填写学校名
  4. 关掉VPN！用国内直连提交
  5. GitHub 发验证邮件 → 点链接确认 → 等待审核（通常几小时）
⚠️ 常见被拒原因：开了VPN（最常见！）、学生证没日期、个人信息和学信网不一致、提交次数过多。关掉VPN + 翻译材料，90%能过。
