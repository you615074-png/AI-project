据彭博社5月22日报道，Cursor的年化收入（ARR）已在4月下旬突破$30亿美元（约¥204亿人民币）。更惊人的是这个增速：

支撑这轮增长的，是大客户的持续涌入。Cursor不再只是个人开发者的玩具，企业团队正在将其作为标准开发工具引入。同月，SpaceX完成对Cursor的收购，Elon Musk将AI编程能力收入囊中。

这背后的逻辑很清晰：生产力工具一旦进入企业工作流，替换成本极高。Cursor卡住了"开发者第二天性"这个位置。

Claude Code：$200/月到底值不值？

Anthropic的终端AI编程代理Claude Code定价$200/月，几乎是GitHub Copilot（$10/月）的20倍。争议随之而来：

  • 支持者说：它能独立处理复杂的多文件重构、自动写测试、理解整个代码库上下文，一个月的产出顶一个初级工程师，$200简直是白菜价
  • 反对者说：终端里的AI代理没有IDE集成，交互体验差。而且——开源世界已经有免费替代品了

Claude Code的真正价值在于多Agent协作视图——你可以在一个界面管理多个并行工作的AI编程代理。这更像是一个"AI工程团队管理面板"，而不仅仅是代码补全工具。

Goose：开源平替的逆袭

Block（原Square）开源的Goose提供了与Claude Code几乎相同的功能——终端AI编程代理，支持多模型切换（Claude/GPT/本地模型），但完全免费。

Goose的杀手锏：

  • ✅ 支持接入任何大模型API（你可以用便宜的DeepSeek）
  • ✅ 完全开源，代码在GitHub可审计
  • ✅ 本地运行，代码不出你的机器
  • ✅ MCP协议支持，可扩展任意工具

对国内开发者来说，Goose + DeepSeek V4的组合可能是性价比最高的AI编程方案。DeepSeek API的价格不到Claude的1/10，而Goose负责编排执行。

GitHub Copilot：仍是性价比之王

在一片$200/月的定价喧嚣中，GitHub Copilot的$10/月显得格外良心。而且：

  • 学生免费：通过GitHub学生包，Copilot Pro直接白嫖（价值¥870/年）
  • 深度IDE集成：VS Code / JetBrains / Neovim 全部原生支持
  • 代码补全延迟最低：这是日常编码中最影响体验的指标

对于大多数国内开发者，Copilot + 一个免费的终端Agent（Goose或Aider）就是最佳组合。
